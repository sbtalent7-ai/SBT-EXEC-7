SBT Executive – Master Data

Ce dépôt statique expose les fichiers maîtres utilisés pour le site SBT Executive.

  • sbt_master_hierarchical.json – structure hiérarchique (pages → blocs → CTA).
  • master_with_cta.csv – source CSV fusionnée avec tous les CTA.

Ces fichiers peuvent être consommés depuis un frontend (Next.js, React, etc.) ou une API route Vercel.
